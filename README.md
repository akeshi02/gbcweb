# Pokémon Coral Version — Browser Emulator

Play **Pokémon Coral Version (2022 Demo v1.2)** directly in your browser using the WasmBoy Game Boy Color emulator.

## 🎮 Controls

| Key | Action |
|-----|--------|
| Arrow Keys | D-Pad |
| Z | A Button |
| X | B Button |
| Enter | Start |
| Backspace | Select |
| F | Fullscreen |

Mobile touch controls are also supported via the on-screen buttons.
